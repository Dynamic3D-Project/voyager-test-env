Getting Started with Voyager


Welcome to the Smithsonian Voyager Explorer 3D web viewer. Before you begin exploring the model, familiarise yourself with the navigation possibilities by taking a tour that explains the various features of Voyager via the menu bar to the left-hand side of the 3D viewer.

To begin this tour, click on the World Icon and select the tour titled "Voyager Quick Start Guide" for a short guided introduction on how to use the interface for exploring this 3D model.
